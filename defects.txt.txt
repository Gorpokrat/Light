Баг-репорт: Класс StringUtils

1. Метод: capitalize()
   Описание: Метод не обрабатывает строки, состоящие только из цифр. Например, входная строка "123abc" возвращает "123abc", что может быть неожиданным поведением.
   Ожидаемое поведение: Метод должен возвращать строку без изменений, если она состоит только из цифр.

2. Метод: trim()
   Описание: Метод не удаляет пробелы в середине строки. Например, входная строка "  hello world  " возвращает только пробелы с начала и конца.
   Ожидаемое поведение: Метод должен удалять пробелы с обеих сторон строки.

3. Метод: contains()
   Описание: Метод не учитывает регистр символов. Например, входная строка ("SkyPro", "s") возвращает False, хотя символ 's' присутствует в строке в верхнем регистре.
   Ожидаемое поведение: Метод должен быть нечувствителен к регистру.

4. Метод: delete_symbol()
   Описание: Метод не удаляет символы, если они находятся в верхнем регистре. Например, входная строка ("SkyPro", "p") возвращает ту же строку.
   Ожидаемое поведение: Метод должен удалять все вхождения символа независимо от регистра.
